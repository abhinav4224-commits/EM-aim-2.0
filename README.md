## Earnings Manipulation Prediction App

This Streamlit app predicts the likelihood of earnings manipulation
using financial ratios based on the Beneish model.

### Inputs
- DSRI
- GMI
- AQI
- SGI
- DEPI
- SGAI
- ACCR
- LEVI

### Output
- Low Risk / High Risk classification
- Probability score

### Tech Stack
- Python
- Streamlit
- Scikit-learn
